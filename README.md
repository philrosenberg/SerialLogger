This is a really basic command line application to log serial data on a Windows machine.

I only provide build files for Visual Studio. Sorry. But it's easy to create your own as it only depends on system libraries.

It's really useful for science instruments as it is generic and easy to use. It timestamps data at specific intervals.

You will need to grab my header only scientific library, as I use my time and unicode functions from this code. You can get it from https://github.com/philrosenberg/sci, just put the headers somewhere they can be found.
